# Proyecto 1  Sistema de Inventario de Tienda de ropa 

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de una aplicación en Java para la gestión de inventario de una tienda de ropa. 
El sistema permite registrar productos, buscar productos, eliminar productos, registrar ventas y generar reportes en formato HTML.

El objetivo del proyecto es aplicar los conceptos básicos de programación, manejo de archivos y desarrollo de interfaces gráficas.

## Funcionalidades del Sistema

Agregar productos al inventario
Buscar productos por ID
Eliminar productos
Registrar ventas
Generar reporte de inventario en HTML
Generar reporte de ventas en HTML
Visualizar datos del estudiante



## Tecnologías Utilizadas

Java, NetBeans IDE, Git y GitHub
